# CalorieFlow

CalorieFlow is a sleek and efficient calorie calculation application built with Flutter. It helps users track their daily caloric intake based on their physical profile and activity level.

## Features
- **Accurate BMR Calculation:** Uses Mifflin-St Jeor formulas for precise calorie estimation.
- **Activity Level Integration:** Calculates TDEE (Total Daily Energy Expenditure) based on individual activity levels.
- **Responsive UI:** Modern, dark-themed user interface built with Flutter Material 3.
- **DevOps-Powered:** Automated CI/CD pipeline using GitHub Actions for seamless building and deployment.

## Tech Stack
- **Framework:** Flutter
- **Language:** Dart
- **CI/CD:** GitHub Actions
- **Deployment:** GitHub Pages

## DevOps Practices
- **Version Control:** Managed with Git, following feature-branch workflows.
- **Automation:** Automated build processes triggered by commits to the `main` branch.
- **Workflow:** Strictly organized merge strategies and branch management to ensure codebase stability.
